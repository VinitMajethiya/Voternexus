# VoterNexus v2.0

VoterNexus is an inclusive, AI-powered civic technology platform designed to empower citizens through accessible information, fact-checking, and interactive civic tools.

## Features
- **Universal Accessibility Suite:** Features designed for visual, motor, and cognitive inclusivity (e.g., text-to-speech, high-contrast, dyslexia-friendly fonts).
- **Gemini-Powered AI Tools:** Includes a Misinfo Shield for real-time fact-checking and an Affidavit Decoder for comparing political candidates.
- **D-Day Simulator:** Interactive timeline and checklist to prepare citizens for voting day.
- **Multilingual Support:** Localized across major Indian languages ensuring broad accessibility.
- **Interactive UI/UX:** Built with Framer Motion, Next.js, and Tailwind CSS for a premium user experience.

## Tech Stack
- **Framework:** Next.js 14
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **AI Integration:** Google Generative AI (Gemini), Anthropic Claude
- **Cloud APIs:** Google Cloud Text-to-Speech, Google Cloud Translate
- **State Management:** Zustand
- **Internationalization:** next-intl

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Setup environment variables in a `.env.local` file:
   - `GEMINI_API_KEY`
   - `GOOGLE_CLOUD_PROJECT`
4. Run the development server: `npm run dev`

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
